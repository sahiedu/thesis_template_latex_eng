# Thesis Template LaTeX (ENG)

This is a clean and well-structured LaTeX template based on the Legrand Orange Book for writing academic theses.
It includes chapters, bibliography and title page.

## Text Styling
```latex
\textbf{Bold}, \textit{Italic}, \underline{Underline}, \texttt{Monospaced}, \emph{Emphasized}
```

## Quotations
```latex
This is an inline quote: ``This is quoted.''

\begin{quote}
	This is a block quote. It's often used for long quoted passages, interviews, or highlighted text.
\end{quote}
```

## Footnotes
```latex
This is a footnote\footnote{Footnote example...}.
```
